📚 COMPLETE API ENDPOINT LIST
=============================

USERS:
  GET    /users                    List all users
  POST   /users                    Create new user
  GET    /users/{user_id}          Get specific user
  PUT    /users/{user_id}          Update user
  DELETE /users/{user_id}          Delete user

POLICIES:
  GET    /policies                 List active policies
  POST   /policies                 Create new policy
  GET    /policies/{policy_id}     Get specific policy
  GET    /policies/type/{type}     Get policies by type

RECOMMENDATIONS:
  POST   /recommendations          Create recommendation
  GET    /recommendations/user/{user_id}  Get recommendations
  PUT    /recommendations/{id}/accept     Accept recommendation
  PUT    /recommendations/{id}/reject     Reject recommendation

ADVISORY:
  GET    /advisory/{user_id}       Get personalized advice

CLAIMS:
  POST   /claims                   File new claim
  GET    /claims/{claim_id}        Get claim details
  GET    /claims/user/{user_id}    Get user's claims
  PUT    /claims/{claim_id}/status Update claim status


🔧 TROUBLESHOOTING QUICK FIXES
==============================

Problem: "Python not found"
Solution: Install Python from python.org, add to PATH, restart terminal

Problem: "ModuleNotFoundError"
Solution: Run: pip install -r requirements.txt

Problem: "Port 8000 already in use"
Solution: Run: uvicorn main:app --reload --port 8001

Problem: "Database errors"
Solution: Delete insurance.db file and restart server

Problem: "Postman collection won't import"
Solution: Make sure Postman_Collection.json is in the project folder
